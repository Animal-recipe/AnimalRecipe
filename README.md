# AnimalRecipe

### **소개**

반려동물 집사들이 그들의 반려동물을 위해 만든 수제 식사, 간식 레시피를 수집하고

레시피 포스트, 후기, 채팅들을 통해 정보를 공유하며 수제 상품들에 대한 광고나 검색 제공하는 웹사이트입니다.
